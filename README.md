# Signup-Sign-application (C++) 🚀

A robust, console-based User Authentication System built using **Object-Oriented Programming (OOP)** in C++. This project demonstrates the lifecycle of objects and memory management in a professional software environment.

## 🛠️ Key Technical Features
- **OOP Principles:** Implementation of Encapsulation to protect user credentials.
- **Constructor Overloading:** Uses Default, Parameterized, and Copy Constructors for diverse object initialization.
- **Dynamic Memory Management:** Efficient use of the `new` and `delete` keywords for Heap memory allocation.
- **Deep Copying:** Demonstrates the use of a Copy Constructor to duplicate object data safely.
- **Pointer Manipulation:** Uses pointers and the arrow operator (`->`) for dynamic interaction.

## 📂 File Structure
- `Sign up Application.cpp`: The main source code containing the `MUser` class and implementation logic.
- `README.md`: Project documentation.
